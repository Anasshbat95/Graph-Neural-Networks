# Graph-Neural-Networks
GNN-basierte Vorhersage des elektrischen Widerstands polykristalliner Batteriematerialien. Mikrostrukturen werden als Graphen modelliert (Körner = Knoten, Korngrenzen = Kanten). Zwei Architekturen (NNConv &amp; GCN) verglichen. Schnellere Alternative zu klassischen FEM-Simulationen. | JLU Gießen – Projekt Data Analytics WiSe 25/26
